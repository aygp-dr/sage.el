* Public API

** Core Functions

*** gemini-repl-memory-add (key value &optional category)
Add or update a fact. Category defaults to 'general'.

#+begin_src emacs-lisp
(gemini-repl-memory-add "editor" "Emacs" 'preference)
#+end_src

*** gemini-repl-memory-get (key)
Retrieve a fact by key. Returns plist or nil.

#+begin_src emacs-lisp
(gemini-repl-memory-get "editor")
;; => (:key "editor" :value "Emacs" :category preference :timestamp "...")
#+end_src

*** gemini-repl-memory-remove (key)
Remove a fact. Returns t if removed, nil if not found.

#+begin_src emacs-lisp
(gemini-repl-memory-remove "old-fact")
#+end_src

*** gemini-repl-memory-list (&optional category)
List all facts or filter by category.

#+begin_src emacs-lisp
(gemini-repl-memory-list)           ;; All facts
(gemini-repl-memory-list 'project)  ;; Only project facts
#+end_src

*** gemini-repl-memory-clear ()
Clear all facts (prompts for confirmation).

#+begin_src emacs-lisp
(gemini-repl-memory-clear)
#+end_src

*** gemini-repl-memory-to-context ()
Generate formatted context string for LLM.

#+begin_src emacs-lisp
(gemini-repl-memory-to-context)
;; => "# Stored Facts\n\nThe following facts..."
#+end_src

** Persistence Functions

*** gemini-repl-memory-save ()
Manually save facts to disk.

#+begin_src emacs-lisp
(gemini-repl-memory-save)
#+end_src

*** gemini-repl-memory-load ()
Manually load facts from disk.

#+begin_src emacs-lisp
(gemini-repl-memory-load)
#+end_src

* Data Structure

** Fact Format

Each fact is a plist:

#+begin_src emacs-lisp
(:key "editor"
 :value "Emacs"
 :category preference
 :timestamp "2024-12-23T17:00:00-0800")
#+end_src

** Storage Format (JSON)

#+begin_src json
[
  {
    "key": "editor",
    "value": "Emacs",
    "category": "preference",
    "timestamp": "2024-12-23T17:00:00-0800"
  }
]
#+end_src

* Categories

1. *general* - General information about the user
2. *preference* - User preferences (editor, theme, workflow)
3. *project* - Project-specific information
4. *technical* - Technical knowledge and environment details

* Configuration Options

#+begin_src emacs-lisp
;; Storage location
(setq gemini-repl-memory-file
      (expand-file-name "gemini-repl/memory/facts.json" user-emacs-directory))

;; Auto-load facts on first access (default: t)
(setq gemini-repl-memory-auto-load t)

;; Auto-save after modifications (default: t)
(setq gemini-repl-memory-auto-save t)

;; Include memory in conversations (default: t)
(setq gemini-repl-use-memory t)
#+end_src

* Usage Workflow

** Initial Setup

#+begin_src emacs-lisp

(gemini-repl-memory-add "name" "Alice" 'general)
(gemini-repl-memory-add "editor" "Emacs" 'preference)
(gemini-repl-memory-add "os" "FreeBSD" 'technical)
(gemini-repl-memory-add "project" "gemini-repl" 'project)
#+end_src

#+begin_src emacs-lisp

#+end_src

** Review Facts

M-x gemini-repl-memory-list

;; List specific category

#+end_src

** Update Facts

#+begin_src emacs-lisp
;; Update existing fact
(gemini-repl-memory-add "project" "new-project" 'project)

;; Remove outdated fact
(gemini-repl-memory-remove "old-key")
#+end_src

* Integration Points

** Automatic Context Injection

Memory context is automatically injected when:
1. =gemini-repl-use-memory= is t (default)
2. Starting a new conversation (conversation list is empty)
3. There are stored facts available
