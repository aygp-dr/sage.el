* 5-Minute Setup

** Step 1: Load the Module

#+begin_src elisp
(require 'sage-project)
#+end_src

** Step 2: Configure (Optional)

#+begin_src elisp
;; Enable auto-load and auto-save (recommended)
(setq sage-project-auto-load t)
(setq sage-project-auto-save t)

;; Set storage directory (optional, defaults to ~/.emacs.d/sage/projects)
(setq sage-project-directory
      (expand-file-name "sage/projects" user-emacs-directory))
#+end_src

** Step 3: Start Using

Open a file in your project and start the REPL:

#+begin_src elisp
M-x sage
#+end_src

Your conversation will automatically be saved per-project!

* Common Commands

** Interactive Functions

| Command                              | Description                    |
|--------------------------------------+--------------------------------|
| =M-x sage-project-load=       | Load project conversation      |
| =M-x sage-project-stats=      | Show conversation stats        |
| =M-x sage-project-clear=      | Clear (and archive) conversation |
| =M-x sage-project-list-archives= | List archived conversations |

** Programmatic Usage

#+begin_src elisp
;; Append a message
(sage-project-append '("user" "How do I fix this bug?"))

;; Archive current conversation
(sage-project-archive "before-major-refactor")

;; Export to Markdown
(sage-project-export "conversation.md" 'markdown)

;; Set custom metadata
(sage-project-metadata :task_id "PROJ-123")
#+end_src

* File Locations

Your conversations are stored at:

#+begin_example
~/.emacs.d/sage/projects/-home-user-myproject/
├── conversation.jsonl   # Current conversation
├── metadata.json        # Project info
└── history/            # Old conversations
    └── 20241223-140530.jsonl
#+end_example

* Working with Multiple Projects

Simply change directories - the conversation switches automatically:

#+begin_src bash
cd ~/project-a
emacs file.py  # Loads project-a conversation

cd ~/project-b
emacs main.rs  # Loads project-b conversation
#+end_src

Each project has its own isolated conversation history.

* Archive Your Conversations

#+begin_src elisp
;; Manual archive with custom name
(sage-project-archive "sprint-42-completion")

;; Auto-archive happens when you exceed max-messages (default: 1000)
;; Configure with:
(setq sage-project-max-messages 500)
#+end_src

* Export and Share

#+begin_src elisp
;; Export to JSON
(sage-project-export "conversation.json" 'json)

;; Export to Markdown for documentation
(sage-project-export "conversation.md" 'markdown)
#+end_src

* Track Custom Metadata

#+begin_src elisp
;; Store git branch
(sage-project-metadata :git_branch "feature/new-api")

;; Store task/issue ID
(sage-project-metadata :task_id "PROJ-123")

;; Any custom field
(sage-project-metadata :sprint "Sprint-42")

;; Retrieve metadata
(sage-project-metadata :git_branch)  ; Returns "feature/new-api"
#+end_src

* Troubleshooting

** Conversation Not Loading?

#+begin_src elisp
;; Check project directory detection
(sage-project-dir)

;; Manually set project directory
(sage-project-set-dir "/path/to/project")
#+end_src

** Want to Start Fresh?

#+begin_src elisp
;; Clear conversation (archives it first)
(sage-project-clear)
#+end_src

** Need to Find Old Conversations?

#+begin_src elisp
;; List all archives
(sage-project-list-archives)

;; Load an archive
(sage-project-load-archive "20241223-140530.jsonl")
#+end_src

* Advanced: Integration with Hooks

#+begin_src elisp
;; Auto-save git context when starting REPL
(defun my-save-git-context ()
  "Save git commit and branch to metadata."
  (when-let ((rev (vc-git-working-revision default-directory)))
    (sage-project-metadata :git_commit rev)
    (when-let ((branch (vc-git--symbolic-ref default-directory)))
      (sage-project-metadata :git_branch branch))))

(add-hook 'sage-mode-hook #'my-save-git-context)
#+end_src

* Example: Daily Workflow

#+begin_src elisp
;; Morning: Start work on project
(cd "~/my-project")
(sage)  ; Loads yesterday's conversation automatically

;; Throughout the day: Chat with AI
;; All messages are automatically saved

;; Before major refactor: Archive current state
(sage-project-archive "before-refactor")

;; End of day: View stats
(sage-project-stats)
;; => Project: /home/user/my-project
;; => Messages: 42 (user: 21, assistant: 21)
;; => Size: 15234 bytes

;; End of week: Export for documentation
(sage-project-export "weekly-notes.md" 'markdown)
#+end_src
